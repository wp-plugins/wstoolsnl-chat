Chat direct met uw bezoekers met de live chat support plugin van WStools.nl

== Description ==

Met de chat module plugin voor WordPress kunt u de chat functie van WStools.nl gemakkelijk integreren in uw WordPress website.
Voor deze plugin is een geregistreerde account nodig van [WStools.nl](http://wstools.nl/).

= Openingstijden beheren =
In het beheer gedeelte van WStools.nl kunt u zelf aangeven welke dagen u open bent. U kunt zelf de tijd instellen en uw chat voorzien van eigen teksten. Het is mogelijk om een bezoeker een automatische chat uitnodiging te sturen na x aantal bezochte pagina's

= Bezoekers uitnodigen =
U kunt bezoekers op uw website een uitnodiging sturen. De bezoeker krijgt een popup om een chat te openen.

= Bezochte pagina's =
U kunt zien welke pagina's de bezoeker heeft bezocht, hoelang de bezoeker al online is en waar de bezoeker zich op dat moment bevindt.

== Installation ==

U dient alleen de plugin te activeren. Mocht u de positie willen aanpassen dient u een div met id WStoolschatcon aan te maken.
